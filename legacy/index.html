LeSpot

AI APP · FULL-STACK

Free cultural event discovery for international students in Paris. 1,500+ curated events across all 20 arrondissements — surfaced by an AI layer that matches your mood, schedule, and interests in seconds. The frontend is a React + TypeScript app deployed on Vercel; the backend runs on Supabase (PostgreSQL with row-level security and real-time sync); Gemini AI handles semantic mood-to-event matching. Built in collaboration with Open Data University and Ministère de la Culture. Live beta.

AI AGENTS · INFRA OpenClaw Agents

OpenClaw is a personal AI agent framework for coordinating teams of specialised LLM agents around a shared goal. Each agent holds a distinct role and communicates through a structured REST API layer, enabling autonomous delegation and multi-step task execution without human intervention. The stack runs on Python and AWS Lambda, orchestrated via Anthropic's Claude API for reasoning and inter-agent messaging.

Python · AWS Lambda · REST API · Claude API

Applied to: multi-agentic office infrastructure · fully integrated outreach teams · idea testing & research frameworks.
